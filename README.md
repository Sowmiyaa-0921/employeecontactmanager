📱 Employee Contact Manager App
📌 Project Description

The Employee Contact Manager is an Android application developed using Java and XML in Android Studio.
The app allows users to enter employee details and manage contact information in a simple and organized way.
This application demonstrates proper UI design, form handling, dynamic layouts, and Android event handling concepts.

🎯 Objective

The main objective of this project is to:

Collect employee details (Name, ID, Department)
Display saved employee information dynamically
Provide quick action buttons like calling and sending email
Maintain proper alignment and professional UI design

🛠️ Technologies Used

Java

XML (Android Layout Design)
Android Studio
GridLayout & LinearLayout
Intent (Dial & Email Functionality)

✨ Features

✔ Clean and professional UI design
✔ Left-side labels with right-side input fields
✔ Department selection using Spinner
✔ Save button to store employee details
✔ Dynamic two-column GridLayout display
✔ Dial Accountant button
✔ Send Email button
✔ Responsive layout with proper spacing

📱 How It Works

User enters:

Name
Employee ID
Department
When the Save button is clicked:
The entered details are displayed in a structured 2-column layout.
Multiple employee entries can be added dynamically.
Bottom action buttons:
Dial Accountant → Opens phone dialer
Send Email → Opens email app with predefined recipient

📂 Project Structure

activity_main.xml → UI Design
MainActivity.java → Application Logic
GridLayout → Dynamic Employee Display
